# Images Folder

This folder is for storing images used in the website.

## Recommended Images to Add:

### Logo
- `logo.png` - Main logo (transparent background recommended)
- Size: 200x200px or similar

### Hero/Banner Images
- `hero-banner.jpg` - Main hero section background
- Size: 1920x1080px

### Service Icons/Images
- `blood-test.jpg` - Blood test image
- `lab-equipment.jpg` - Laboratory equipment
- `technician.jpg` - Lab technician at work
- `report.jpg` - Medical report image

### About Section
- `lab-interior.jpg` - Laboratory interior
- `team.jpg` - Team photo
- `equipment.jpg` - Modern equipment

### Testimonials (Optional)
- `patient1.jpg` - Patient testimonial photo
- `patient2.jpg` - Patient testimonial photo

### Certifications
- `nabl-certificate.jpg` - NABL certification
- `iso-certificate.jpg` - ISO certification

## Image Guidelines:

1. **Format**: Use JPG for photos, PNG for logos/icons
2. **Size**: Optimize images (max 500KB per image)
3. **Dimensions**: 
   - Logo: 200x200px
   - Hero: 1920x1080px
   - Service cards: 800x600px
   - Icons: 100x100px

4. **Optimization Tools**:
   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - ImageOptim (Mac)

## How to Add Images:

1. Save your images in this folder
2. Update the HTML files with correct image paths
3. Example: `<img src="images/logo.png" alt="Blood Care Logo">`
